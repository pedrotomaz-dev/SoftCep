# SoftCep API

**SoftCep** é uma API REST intermediária desenvolvida em **.NET 8**, que atua como camada anti-corrupção entre aplicações clientes e o serviço externo **ViaCEP**.  
Ela fornece uma interface confiável para consultas de CEP por código numérico ou endereço, garantindo resiliência, desempenho e manutenibilidade.

## Objetivo do Projeto

Este projeto foi desenvolvido como parte de um **desafio técnico**, com foco em:

- Aplicar **boas práticas de arquitetura** (DDD, SOLID, Clean Architecture)
- Demonstrar **integração com serviços externos** (ViaCEP)
- Implementar **testes automatizados** (xUnit)
- Utilizar **Docker** para empacotamento e deploy
